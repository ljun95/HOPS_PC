회원 가입 및 확인 (2/4)

○ 인증창이 표지 되지 않는 경우, 이렇게 해결하세요.

※ Internet Explorer 설정

  1. 상단 메뉴 [도구 > 인터넷 옵션 ] 선택
  2. [인터넷 옵션 > 보안 > 인터넷 아이콘] 선택
  3. 보안 수준이 [높음] 이라면 보안 수준을 [보통]으로 설정
  4. [ □ 보호 모드 사용] 의 체크를 해제
  5. [인터넷 옵션 > 보안 > 신뢰할 수 있는 사이트 아이콘] 선택
  6. [사이트] 클릭 후, 신뢰할 수 있는 사이트에
    [http://000,000.or.kr] 입력 후 [추가] 클릭
    [https://000.000.or.kr] 입력 후 [추가] 클릭
  7. 보안 수준이 [높음]이라면, 보안 수준을 [보통]으로 설정
  8. [□ 보호 모드 사용]의 체크를 해제
  9. 인터넷 옵션 팝업창의 [확인]을 클릭하여 종료
