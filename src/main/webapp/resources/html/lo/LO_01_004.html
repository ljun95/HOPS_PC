비밀번호 재설정

회원님의 개인정보 보호를 위해 비밀번호를 변경하여 주시기 바랍니다.

  • 비밀번호 생성 시 영문자, 소문자, 특수문자 중 3가지 이상 조합 시 8자리 이상 또는 2가지 문자 조합 시 10자리 이상으로 권고함
  • 특수문자 중 -,…, //(하이픈, 마침표, 슬래시 연달아 2개)는 사용 불가
  • 숫자, 영문자의 조합으로 8~12자리로 입력해주세요
  • 연속된 문자열 (123또는 4321. abcd, dcba 등)을 4자 이상 사용 할 수 없습니다.
  • 이전 비밀번호는 다시 사용 불가 합니다.
